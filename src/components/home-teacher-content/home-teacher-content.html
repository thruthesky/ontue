Steps you need to do to become a teacher.

  1. Install Kakaotalk app. ( THIS IS A MANDATORY REQUIREMENT )
  2. Register
  3. Complete profile update with all your data.
    • You will need to upload nice photo
    • You will need to upload a short introduction video on youtube
  4. Add your schedule.
  5. Contact to admin to get higher grade. ( Important Note: Teacher must use smart phone to add contact to admin. Once teacher added contact to admin, teacher can use PC to chat with admin. ) Admin will check your environment.
    • Good audio/video device
    • Fast speed of internet
    • Calm space to conduct the lesson
  6. Update your payment information so that you can get your pay.
* Important Tips !!
- Teacher must add student using smart phone.
- There is no way to add friend in kakao with desktop software.
* Teachers Table
- Teacher without grade will not be shown if Recommended teacher is selected.
- But they will be shown if the option "Show all teachers" is selected.

INSTALLING KAKAOTALK ON DESKTOP

WHY KAKAOTALK?

Most of the student are from Korea, and most of them have kakaotalk. And it would be more convenient for them to have class with kakaotalk app.

Download: Kakaotalk Installer

1. Install Kakaotalk App from Google Play(android) or AppStore(iphone) to your smart phone

2. After installation. Open Kakaotalk app.

3. Select your country code and enter your phone number, and press Next button.

4. Kakaotalk will send you a sms containing the 4-digit verification code. Input the code and press the Next button.

  • *If you are using your number on the same device it will automatically add the verification number.
  • *If not then you need to type the 4-digit verification code that you will received.

5. Press Sign Up Button

6. Enter your name, upload your photo and press get started.

7. Viola! You Successfully Installed and Created your kakaotalk Account.
